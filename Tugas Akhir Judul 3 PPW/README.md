# Tugas AKhir Percobaan 3 Tailwind - QuantumLeap Solutions - Nexus Platform

Website ini adalah landing page resmi untuk QuantumLeap Solutions, menampilkan platform manajemen proyek berbasis AI bernama Nexus. Website ini dirancang dengan tema dark modern, responsif, dan sepenuhnya menggunakan Tailwind CSS.

## Deskripsi Proyek

Nexus adalah platform manajemen proyek cerdas yang memanfaatkan AI untuk mengotomatiskan tugas, memprediksi deadline, dan meningkatkan kolaborasi tim. Website ini menampilkan fitur utama Nexus, misi perusahaan, form kontak, serta navigasi yang mudah digunakan baik di desktop maupun mobile.

### Fitur Utama Website
- Navigasi responsif (navbar & sidebar)
- Hero section dengan headline dan CTA
- 3 fitur utama Nexus
- About section (cerita perusahaan)
- Form kontak
- Footer dengan social media dan navigasi
- Gradient background di setiap section
- Semua konten dalam Bahasa Indonesia

## Instalasi

1. **Clone atau Download Repository**
   - Download folder `Tugas Akhir Judul 3 PPW` ke komputer Anda.

2. **Pastikan File Berikut Ada**
   - `index.html` (file utama website)
   - `README.md` (file ini)
   - (Opsional) `styles.css` jika ingin menambah custom CSS

3. **Tidak Perlu Instalasi Khusus**
   - Website ini hanya membutuhkan browser modern (Chrome, Firefox, Edge, Safari)
   - Tailwind CSS sudah di-load via CDN di dalam `index.html`

## Cara Penggunaan

1. **Buka Website**
   - Klik dua kali `index.html` atau buka dengan browser favorit Anda.

2. **Navigasi**
   - Gunakan menu di header atau sidebar (mobile) untuk berpindah antar section.
   - Klik tombol di footer untuk kembali ke atas (hero section).

3. **Form Kontak**
   - Isi form kontak untuk mengirim pesan ke tim QuantumLeap Solutions.

4. **Customisasi**
   - Untuk mengubah konten, edit langsung file `index.html`.
   - Untuk menambah style, tambahkan di `styles.css` (jika ada).
